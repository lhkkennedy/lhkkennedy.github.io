Public Debt

The UK owes a lot of money

Understanding our public debt Data Sciece project

Governments across the globe in response to the COVID-19 pandemic have enacted massive spending packages financed by a lot of public debt. A question often asked is: how is the government able to borrow so much cash and can the tax payers pay it back


Why does the UK owe £2.2 trillion or roughly £32,000 per person in public sector debt ? That's 103.6% of GDP for the 2021 financial year(ONS, 2021). How can the government afford to acrew massive amounts of debt and is there a limit? This project explores through a number of visualisations the state of the UK's public sector debt.

How much are we borrowing?

The UK government has the capacity if it needs, to spend more than it can raise in tax revenue such as in a crisis and finance spending through public borrowing. 2020 has seen record levels of borrowing as almost doubles the borrowing in 2009 during the financial crash. In the 2020/2021 financial year the UK had borrowed £272.879 Billion


The current budget deficit and borrowing are correlated however, you can also see that borrowing had not yet completely tapered off following the 2008 financial crash. The fall in the deficit is the result of austerity measures shrinking the public sector post crash as there was fears about the ballooning debt levels.

Is that a lot?

Levels of public debt last seen in early 1960s
You can see that borrowing had not completely tapered off after the 2008 financial crash. The government did not have a chance to start servicing this debt as austerity measures meant to help balance the budget induced less spending and growth; culminating in less tax revenue to service the debt.
Leading governments have continued to borrow and spend at record levels as they and central banks have begun adopted new thinking about how much they are able to borrow sustainably. Central banks have supported the governments fiscal efforts by increasing the money supply. And an increase in saving rates have kept interest rates low. With record low interest rates the past two decades it has been very easy for GDP to outpace the interest payments of debt.

Where does the UK compare to the rest of the world in debt-to-GDP ratio?
The countries with highest debt-to-gdp ratios are countries such as Venezuela which holds 350% of GDP in public debt - Japan at 266% of GDP - Sudan at 259% and Greece at 206%. That means the government in these countries owe more than the equivalent of every finished good and service produced in the country in a year twice. More than three times in the case of Venezuela
The UK can be seen 30th on the list and the US is 11th. Although the UK has been increased its borrowing massively due to the size of GDP, the Debt-to-GDP ratio does not reach levels like Greece or Venezuela.

Scraper

A python script was created to scrape the country name and their respective debt to gdp ratios that was displayed in a table at https://tradingeconomics.com/country-list/government-debt-to-gdp. The data was placed in a dataframe then formatted and exported as a csv file. 
   import pandas 
   as pd
   import requests
   import os
   from bs4 import BeautifulSoup
   
   #downloading the source code for the URL then searching for a table
   URL = "https://tradingeconomics.com/country-list/government-debt-to-gdp"
   html = requests.get(URL)
   soup = BeautifulSoup(html.content, 'html.parser')
   table = soup.find_all("table", class_="table table-hover table-heatmap")
   #searching through the tables source code to find the rows.
   table_rows = table[0].find_all('tr')
   
   df = pd.DataFrame()
   # I iterate through the array of rows and extract the country names and values
   # Countries with multiple words required special treatment 
   # They would take up multiple places in the row array shifting the value place
   for tr in table_rows:
       s = table_rows.index(tr)
       items = tr.text
       # Split the text elements in each row text into an array
       arr = items.split()
       extra = len(arr) - 5
       country = arr[0]
       value = arr[1+extra]
       # If the coutntry name is longer than one word we loop until all words are joined.
       for x in range(extra):
           country = country + ' ' + arr[x+1]
       results = [country, value]
       print(results)
       # The data is placed into a data frame to be formated
       df_t = pd.DataFrame(results)
       df_t.columns = ['Data']
       df[s] = df_t['Data']
   
   # Transpose and remove the first row.
   df = df.T
   df = df.iloc[1: , :]
   df.columns = ['Country', 'Debt-to-GDP']
   print(df)
   
   #save to csv
   filename = 'Debt-to-GDP.csv'
   abspath = os.path.abspath(__file__)
   dname = os.path.dirname(abspath)
   os.chdir(dname)
   filelocation = os.getcwd()
   print("Saving as:  [ " + filename + " ]  in:  [ " + filelocation + " ] ")
   df.to_csv(filename)

Public Sector Current Budget Deficit and Net Borrowing

December 2020 or 2021 data
Accessed on January 5th, 2021
Time series data was sourced the from ONS: https://www.ons.gov.uk/economy/governmentpublicsectorandtaxes/ publicsectorfinance#timeseries
The dataset was reshaped to a long format with STATA.
The command:
reshape long @milliongdp, i(date) j(type) string
Before plotting this data I ran a python script to convert the date format from e.g. 2001 Q1 tp 2001-01-01 The code is as below: 
import pandas as pd
import os


abspath = os.path.abspath(__file__)
dname = os.path.dirname(abspath)
os.chdir(dname)


filename = "DebtGDP.csv"
df = pd.read_csv(filename)
qs = df['date'].str.replace(r'(\d+) (Q\d)', r'\1-\2')
df['date'] = pd.PeriodIndex(qs, freq='Q').to_timestamp()

filelocation = os.getcwd()
print("Saving as:  [ " + filename + " ]  in:  [ " + filelocation + " ] ")
df.to_csv(filename)
